<h1 >Trends in Startups | codecademy.com Project </h1>

## About The Project

In this project, I analyzed a dataset containing information about various startups. The dataset included details such as the company name, location, category, number of employees, funding raised, valuation, founding year, stage, CEO, and additional information.

## How it works

1. Data Analysis: Utilized SQL queries to extract meaningful insights from the dataset.
2. Visualization: Presented the findings in a clear and concise manner, enabling easy interpretation of trends and patterns in the startup ecosystem.

## Built With

1. SQL: Leveraged SQL queries to manipulate and analyze the dataset efficiently.

### Challenges

1. Data Interpretation: Understanding the nuances of the dataset and formulating queries to extract relevant insights posed a challenge initially.
2. Optimization: Ensuring the SQL queries were optimized for performance while retrieving desired results.

### Learnings

1. SQL Proficiency: Enhanced proficiency in SQL query writing, including aggregation, filtering, grouping, and ordering.
2. Data Interpretation: Developed skills in interpreting and analyzing datasets to derive actionable insights.
3. Presentation: Improved ability to present findings in a structured and comprehensible format.
